# Binary Search

Binary Search is used on sorted arrays.

Algorithm:

1. Find middle element.
2. Compare with target.
3. Search left half.
4. Search right half.

Complexity:

Time: O(log n)

Space:
Iterative: O(1)
Recursive: O(log n)

Common Mistakes:

- Using binary search on unsorted arrays.
- Overflow in mid calculation.

Applications:

- Search in sorted arrays.
- Lower Bound.
- Upper Bound.
