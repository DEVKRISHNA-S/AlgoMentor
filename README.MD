# AlgoMentor

A local AI-powered DSA mentor built using:

- Streamlit
- Ollama
- VibeThinker-3B
- Python
- uv

## Features

- DSA concept explanations
- LeetCode guidance
- Codeforces guidance
- Local LLM execution

## Setup

1. Clone repository
2. Install dependencies
3. Install Ollama
4. Import VibeThinker model
5. Run Streamlit
